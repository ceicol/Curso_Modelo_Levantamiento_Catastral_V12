# Validaciones temáticas

Este conjunto de reglas tienen como propósito asegurar que los datos en campo cumplan con las reglas de negocio definidas, en este orden de ideas, cada gestor catastral puede llegar a solicitar reglas temáticas para asegurar que los datos recopilados en campo cumplan con su lógica de negocio.
